Instructions

Game Duration: 1 minutes

Two Way To End Game:

  1. One minute is up
  2. One of the players earned 1000 points

Initial Player Attributes :

  1. Scores: 0
  2. Player's Attack Monster Scores: 100 (Score earned after killing a monster)
  3. Speed of movement: 200
  4. Player Attack Range: 60

Game Flow :

Pair Up

  1. Press either "Player 1" button or "Player 2" button
  2. Wait for other Player to join the game

Game Play

  1. User can collect gems and swords
  2. User can attack monster and attack other players
  3. When a player is attacked, points are deducted. The player's score will not go below 0

Game Characters

  1. Collect Gem : + 20 Scores in player'Scores
  2. Collect Swords : + 50 in Player's Attack Monster Scores
  3. Kill Monsters : + Player's Attack Monster Scores in player'Scores
  4. Attack Player : Attacker: + 50 Scores, Defender: - 50 Scores

Player Movements :

  1. Move: Arrow key ↑:"Up", ←:"Left", ↓:"Down", →:"Right"
  2. Attack: Z

Cheat Modes :

Click on Keyboard Space Key

  1. Speed of movement: 350
  2. Player Attack Range: 120

End Game :

After finishing the game, there are two buttons to select

  1. Click on "Restart" Button : back to the pair up page
  2. Click on "Back Home" Button : back to the home page
